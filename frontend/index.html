Global Istio Gateway

Global Istio Gateway is a project aimed to deploy istio as an anycast ingress gateway for apps globally. The idea is loosely inspired by services like Cloudflare, and Fastly.

The Idea, in simple terms, is that we would have multiple instances of kubernetes running in various cloud and data centers, hosting istio gateways. Users would register their services on a dashboard. These would be registered as dns services. In return they would be given a publically accessable domain(like MyProject.global.backendengineer.net).
